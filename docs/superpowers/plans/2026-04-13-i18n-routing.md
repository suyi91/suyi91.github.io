# i18n 双语路由 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为博客添加中英双语支持，英文为默认语言（无前缀，`/blog/post`），中文加 `/zh/` 前缀（`/zh/blog/post`），UI 文字和文章内容均随语言切换，语言偏好持久化到 localStorage，无对应翻译时自动回落到另一语言。

**Architecture:** 使用 Astro 5 内置 i18n 路由（`astro.config.mjs` 配置 `i18n.defaultLocale="en"` + `locales=["en","zh"]`）；content collections 按语言子目录组织（`blog/en/`、`blog/zh/`）；i18n 工具函数改为运行时接收 `lang` 参数；Header 增加语言切换按钮，逻辑复用 `toggle-theme.js` 的 localStorage 模式。

**Tech Stack:** Astro 5 i18n routing, Astro content collections, TypeScript, localStorage

## Task 2：改造 i18n 工具函数为运行时参数化

**Files:**

**背景：** 当前 `t` 是编译时固定到 `config.lang` 的，需要改为每个页面/组件可以传入运行时 `lang` 参数。

- [ ] **Step 1: 修改 `src/i18n/utils.ts`**

```ts
import { en } from './en'
import { zhCn } from './zhCn'
import { cs } from './cs'
import { config } from '../consts'

export type Lang = 'en' | 'zh'

const ui = {
  en,
  zh: zhCn,
  cs,
}

export function getLangFromUrl(url: URL): Lang {
  const [, lang] = url.pathname.split('/')
  if (lang === 'zh') return 'zh'
  return 'en'
}

export function useTranslations(lang: Lang) {
  return function t(key: string): string {
    return ui[lang]?.[key] ?? ui['en'][key] ?? key
  }
}

// 向后兼容：默认使用 config.lang 对应的语言
export const t = useTranslations(config.lang === 'zh-cn' ? 'zh' : 'en')
```

## Task 3：客户端语言切换脚本

**Files:**
- Create: `public/toggle-lang.js`

- [ ] **Step 1: 创建 `public/toggle-lang.js`**

```js
// 从 localStorage 读取语言偏好，初始化 <html lang="...">
(function () {
  const savedLang = localStorage.getItem('lang')
  const currentPath = window.location.pathname
  const isZhPath = currentPath.startsWith('/zh')
  const currentLang = isZhPath ? 'zh' : 'en'

  // 若 localStorage 有偏好且与当前路径语言不符，则跳转
  if (savedLang && savedLang !== currentLang) {
    if (savedLang === 'zh' && !isZhPath) {
      window.location.replace('/zh' + currentPath)
    } else if (savedLang === 'en' && isZhPath) {
      window.location.replace(currentPath.replace(/^\/zh/, '') || '/')
    }
  }
})()
```

> 注意：此脚本在 `<head>` 最前面内联执行，防止语言闪烁。

- [ ] **Step 3: 修改 `src/utils/getCollectionByName.ts` 支持按语言过滤**

```ts
import { getCollection } from 'astro:content'
import type { Lang } from '../i18n/utils'

export const getCollectionByName = async (name: string, lang?: Lang) => {
  let posts = await getCollection(name)
  if (!posts || posts.length === 0) return []

  posts = posts.filter(({ data }) =>
    import.meta.env.PROD ? !data.draft : true
  )

  if (lang) {
    // slug 格式为 "zh/post-name" 或 "en/post-name"
    const langPosts = posts.filter((p) => p.slug.startsWith(lang + '/'))
    // 若该语言无内容则回落到另一语言
    return langPosts.length > 0
      ? langPosts
      : posts.filter((p) => p.slug.startsWith('en/'))
  }

  return posts
}
```
